常用快捷键

Super 键: 键盘上带有 Windows 图标的键

Super: 打开活动程序菜单

Ctrl + Alt + T: 打开命令行终端

Super + L: 锁定屏幕

Super + D: 显示桌面，重按返回原来窗口

Super + A: 显示应用程序菜单，同鼠标单击屏幕左下角九个点

Super + Tab 或 Alt + Tab: 在运行的应用程序间切换Super+箭头：移动窗口位置

Super + 箭头: 移动窗口位置

  • Super + 左箭头: 应用程序窗口贴合屏幕的左边缘

  • Super + 右箭头: 应用程序窗口贴合屏幕的右边缘

  • Super + 上箭头: 应用程序窗口最大化

  • Super + 下箭头: 应用程序窗口恢复至正常大小

Super + M: 打开通知栏，重按关闭

Super + Space: 切换输入法

Alt + F2: 运行控制台

Ctrl + Q 或 Ctrl + W: 关闭应用程序窗口

Ctrl + Alt + 箭头: 切换工作区

  • Ctrl + Alt + 上箭头: 上移工作区

  • Ctrl + Alt + 下箭头: 下移工作区

Ctrl + Alt + Del: 注销

设置环境变量

基于软链接

环境变量的添加路径有两种：

第一种：添加到系统自带环境变量路径：/usr/bin，若用户自定义变量添加到该目录，系统升级后会被重置导致用户自定义变量丢失。

第二种：添加到用户自定义环境变量路径：/usr/local/bin，用户自定义环境变量路径。系统升级后会保留该路径下的环境变量。

因此自定义环境变量一定要添加到/usr/local/bin下，防止系统升级丢失。
